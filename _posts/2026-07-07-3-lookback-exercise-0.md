Well, you are here, so it worked out in the end. But I would like to write a little bit about what I did. I started by doing a Jekyll tutorial on [there webpage](https://jekyllrb.com/docs/step-by-step/01-setup/) to have a general idea of what Jekyll is (never heard of it before). From there onwards I needed to deploy this on github pages. 

I first had some trouble connecting my local git repo to a github repo, but with some help from a LLM I debugged the issues and eventually settled on connecting with a SSH key. After that I had some trouble with (relative) paths while migrating the local blog to github pages. After setting things up correctly I removed as much Liquid from Jekyll as possible, because I think it is confusing for now and I would actually like to focus on writing some HTML and CSS for myself. Time for the next exercise.
